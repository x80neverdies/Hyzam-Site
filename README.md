# 🔴 Hyzam Server — Landing Page

> Página de convite estilizada para o servidor do Discord **Hyzam**, com visual dark, animações e efeitos em vermelho.

---

## ✨ Preview

![Hyzam Landing Page](https://i.imgur.com/placeholder.png)

> Fundo preto · Vermelho sangue · Animações CSS · Zero dependências

---

## 🎨 Features

- **Fundo animado** — grade vermelha com scroll infinito
- **Partículas flutuantes** — geradas via JavaScript puro
- **Título com flicker** — efeito de luz instável no nome do servidor
- **Imagem circular** — com anéis pulsantes e glow no hover
- **Botão com sweep** — preenchimento deslizante ao passar o mouse
- **Animações de entrada** — elementos surgem em cascata com delay
- **Tipografia marcante** — fonte Cinzel (Google Fonts)
- **100% offline-ready** — imagem embutida em base64, sem dependências externas

## 🚀 Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/hyzam-landing.git
   ```

2. Abra o arquivo diretamente no navegador:
   ```bash
   open index.html
   ```

Sem build, sem npm, sem frameworks. Só abrir e usar.
